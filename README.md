# laravel-10-setup
Project setup for Laravel - 10

# 1
```
composer create-project laravel/laravel=10.*.* YOUR_PROJECT_NAME --prefer-dist
```
# 2
```
composer require laravel/ui
```
# 3
```
php artisan ui bootstrap --auth
```
# 4
```
npm install
```
# 5
```
npm run dev
```
# 6
```
npm run build
```
